# Transformation-des-produits-carn-s-au-BF
La viande
La transformation des produits carnés consiste à convertir la viande en produits finis ou semi-finis afin d'améliorer sa conservation, sa qualité et sa valeur commerciale. Elle comprend plusieurs opérations comme le découpage, le broyage, le salage, le fumage, la cuisson ou le séchage. Ces procédés permettent aussi d'assurer la sécurité sanitaire et de diversifier les produits destinés à la consommation.
